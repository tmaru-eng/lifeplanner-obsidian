# LifePlanner クイックスタート

## 目的

LifePlannerは週間計画を中心に、目標→タスク分解→実行までを一貫して管理するための
Obsidianプラグインです。

## はじめに

1. プラグインを開くと週間計画タブが表示されます。
2. 目標ゴールタブで人生目標から週間目標までの階層を作成します。
3. 目標→タスク切り出しタブで目標に紐づくタスクを作成します。
4. 週間計画タブでタスクを今週の枠に配置します。
5. Inboxタブでメモした項目を、目標やタスクへ振り分けます。
6. 演習タブで自己分析や価値観整理のワークを作成します。

## 基本タブ

- 週間計画: 今週の予定とタスク配置
- Inbox: なんでもメモし、各タブへ振り分け
- 目標ゴール: 人生目標〜週間目標の階層管理
- 目標→タスク切り出し: 目標からタスクを生成
- 演習: 自己分析やワークの記録

## 保存ルール

- すべてMarkdownで保存します。
- ファイル名は `LifePlanner - <Type>.md` です。
