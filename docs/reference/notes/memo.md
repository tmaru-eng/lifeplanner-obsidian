# LifePlanner 変更メモ

- 既存運用との互換性は不要。Markdownのみで再設計する。
- ファイル命名は `LifePlanner - <Type>.md` に統一する。
- 主要テンプレート: Weekly / Goals / Tasks / Inbox / Exercises

1. スケジュール・行動管理（日常の運用）
■ 手帳リフィル - Daily.csv

項目:

タイムライン（5:00〜24:00）、優先順位（ABC）、ルーティンチェック

今日のセルフトーク（アファメーション）、今日得たい結果（理想の状態）

メモ欄、昨日の振り返り（良かったこと・やり直せるなら）

想定される使い方:

1日1ページのリフィルとして使用。

朝一番に「今日得たい結果」と「優先タスク」を明確にし、精神状態を整える（アファメーション）。

1日の終わりに「振り返り」を行い、翌日の改善につなげる日次PDCAを回すために使います。

■ 手帳リフィル - Weekly(H).csv

項目:

週間目標、今月のテーマ、ルーティーン行動追跡

役割ごとの目標管理（自分、トレーダー、他役割3・4など）と重点領域（身体、社会、知性、精神）

週間PDCA（ゴール確認、課題抽出、解決策、タスク化）、日々の利損記録

想定される使い方:

**見開き1週間のホリゾンタル（横型）**タイプ。

「7つの習慣」のように、役割（ロール）ごとのバランスを重視した週次計画を立てる母艦です。

週末にPDCA欄を使って今週の課題を深掘りし、次週の行動計画へ落とし込むために使います。

■ 手帳リフィル - Weekly(V).csv

項目:

週間目標、今月のテーマ、ルーティーン、役割ごとの重点タスク

バーティカル（縦型）のカレンダー配列、日々の「今日の一言」「利損」

想定される使い方:

**バーティカル（縦型）**タイプ。Weekly(H)と選択して使用。

H版よりも時系列の流れや、日々の特記事項（一言・損益）を一覧することに特化しています。

■ 手帳リフィル - Monthly.csv

項目:

カレンダー、月間目標、プライベート/ビジネスごとの優先事項（Priority）リスト

前月・翌月カレンダー、インデックス

想定される使い方:

月全体の予定俯瞰。

細かい予定よりも、その月に達成すべき**「重要事項（Big Rocks）」**をビジネスとプライベートに分けて管理します。

2. 目標・プロジェクト管理（中長期視点）
■ 手帳リフィル - 5ヵ年計画.csv

項目:

ビジネス（仕事・収入等）とプライベート（家庭・余暇等）の2軸

1年後、3年後、5年後の状態定義、ブレイクダウン

想定される使い方:

中長期的な人生設計図。ここでの理想像を年次・月次の目標へ落とし込みます。

■ 手帳リフィル - 年間目標.csv

項目:

目標設定、フェーズ詳細

第1〜第4四半期ごとのガントチャート風進捗管理

想定される使い方:

1年間を見通したプロジェクト管理。

大きな目標を月単位のマイルストーンに分解して進捗を追います。

■ 手帳リフィル - 4半期目標.csv

項目:

目標、WBS番号、タスクタイトル

12週間（3ヶ月）分の週次ガントチャート

想定される使い方:

「12週間の年度（The 12 Week Year）」のような、3ヶ月サイクルの集中期間管理。

年間目標をさらに具体的なタスク（WBS）レベルまで分解し、週単位で実行管理します。

■ 手帳リフィル - 目標設定.csv

項目:

価値観/ミッション（なぜやるか？）、目標（何をいつ？）、優先度

具体的なステップ（どのように？）、期限

想定される使い方:

個別の目標に対する企画書。

単に「やる」だけでなく「なぜやるのか（動機）」を明確にし、モチベーションを維持する設計になっています。

3. 自己分析・ライフデザイン（価値観の明確化）
■ 手帳リフィル - 自己分析等.csv

項目:

ミッションステートメント、価値観とその説明

深掘り質問集（過去の成功体験、理想の人物、10年後の自分、自分の葬式での弔辞など）

想定される使い方:

自分の行動指針（憲法）を作成するためのワークシート。

定期的に見返し、自分の軸がブレていないか確認するために使います。

■ 手帳リフィル - 欲しい、やりたい.csv

項目:

Have(持つ)/Do(する)/Be(ある)リスト

余命1年リスト、あと100年生きるならリスト、死ぬまでにやりたいことリスト

想定される使い方:

欲望の棚卸し（バケットリスト）。

制約を外して夢を書き出し、それを目標設定シートへ転記するネタ帳です。

4. ログ・リスト・コレクション（記録とデータベース）
■ 手帳リフィル - トレード日誌.csv

項目:

日時、通貨ペア、移動平均線（25/75/200）の状態（日足〜15分足）

エントリー根拠（Long/Short、Lot、TP/SL）、結果、振り返りコメント

想定される使い方:

FXや株の売買記録。

テクニカル分析（マルチタイムフレーム分析）の状況と、エントリー時の心理や根拠を記録し、トレードスキルを向上させるための専門リフィルです。

■ 手帳リフィル - Inbox.csv

項目:

チェックボックス、内容、分類タグ（欲しいw/買うb/やりたいd/行きたいg、メモN/言葉Q/考えI/価値観V/習慣H）

想定される使い方:

頭の中にある「気になること」をすべて書き出すためのInbox（受信トレイ）。

GTDの手法に基づき、まずはここに書き出してから、スケジュールやタスクへ振り分けます。

■ 手帳リフィル - 読書.csv

項目:

書名、著者、開始日

学び・行動（3点）、メモ・感想

想定される使い方:

読書記録。「読んで終わり」ではなく「行動（Action）」に落とし込む欄が3つあるのが特徴です。

■ 手帳リフィル - 座右の銘.csv / 手帳リフィル - 約束.csv

項目:

座右の銘：誰の言葉か、内容

約束：誰と、何を、完了チェック

想定される使い方:

座右の銘: モチベーションアップのための言葉集。

約束: 口約束や小さな依頼を忘れないための備忘録。

■ 手帳リフィル - FuturePaging.csv

内容:

（空欄または罫線のみと推測）

想定される使い方:

自由記述ページ。将来のビジョンを絵で描いたり、マインドマップを描いたりするフリースペースと思われます。
