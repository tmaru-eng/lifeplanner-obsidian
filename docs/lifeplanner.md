# LifePlanner

LifePlannerはObsidian上で週間計画を中心に、目標→タスク→実行を
一貫して管理するプラグインです。

## タブ構成

### 日常（既定表示）
- 週間計画: 今週のタスク配置とレビュー
- Inbox: メモを収集し、目標/タスク/週間へ振り分け
- アクションプラン: 目標に紐づくタスクの整理・編集
- イシュー: 中長期の課題をカンバンで管理
- 目標: 人生目標〜週間目標の階層管理 (親子でネスト表示)
- 約束: 自分や他者との約束を管理

### 内省
- ミッション: 人生の方向性・存在意義を定義
- 価値観: 判断軸・優先順位を明確化
- Have / Do / Be: 欲しいもの / したいこと / ありたい姿の整理
- 演習: 自己分析や内省ワーク (価値観分析、バケットリスト等)

## 保存ルール

- すべてMarkdownで保存します。
- ファイル名は `LifePlanner - <Type>.md` で統一します。

## 設定

- Default tags: LifePlannerで作成/更新するMarkdownにfrontmatterの`tags`を付与します（カンマ区切り）。

## 操作

- 起動時に週間計画タブを表示します。
- 各タブの上部ナビゲーションボタンで切り替えできます。
- コマンドパレットからも各タブを開けます。
