# LifePlanner

LifePlannerはObsidian上で週間計画を中心に、目標→タスク→実行を
一貫して管理するプラグインです。

## 基本タブ

- 週間計画: 今週のタスク配置とレビュー
- Inbox: メモを収集し、目標/タスク/週間へ振り分け
- 目標ゴール: 人生目標〜週間目標の階層管理 (親子でネスト表示)
- 目標→タスク切り出し: 目標からタスクを生成
- 演習: 自己分析や内省ワーク (H/D/B、バケットリスト、価値観など)

## 保存ルール

- すべてMarkdownで保存します。
- ファイル名は `LifePlanner - <Type>.md` で統一します。

## 操作

- 起動時に週間計画タブを表示します。
- 各タブの上部ナビゲーションボタンで切り替えできます。
- コマンドパレットからも各タブを開けます。
