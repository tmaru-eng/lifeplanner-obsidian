人生を計画・実行するための管理アプリとして、目的・役割を明確化しています。

## 2. 日常（既定表示）

### 週間計画
- **目的**: 今週の行動計画を実行可能な形に落とす
- **内容**
  - 今月のテーマ（同月内で共有）
  - ルーティン行動（週を跨いで共有）
  - 役割と重点タスク（役割は共有、目標は週ごと）
  - 今週の振り返り（良かったこと / 課題）
  - 日付ごとの一言メモ（曜日ごとのメモ）
- **想定フロー**
  1) 役割の観点から今週の目標を記入
  2) ルーティンをチェック
  3) 毎日メモを書き、週末に振り返る

### Inbox
- **目的**: 思いつきを一時的に受け止める
- **内容**
  - メモの追加/編集/削除
  - 後で目標・タスク・週間計画へ移動

### アクションプラン
- **目的**: 目標に紐づくタスクを整理・編集
- **内容**
  - 目標に関連するタスクの登録・完了管理
  - 週間計画側のアクションプラン欄と連動
- **想定フロー**
  1) 目標からタスクを起こす
  2) 週間計画へ割り当てる

### イシュー（Kanban）
- **目的**: 中長期の課題をカンバンで管理
- **内容**
  - 列ベースで課題を追跡
  - 設定でカラム変更可

### 目標
- **目的**: 人生 → 長期 → 中期 → 年間 → 四半期 → 月間 → 週間の階層化
- **内容**
  - ネスト構造、ドラッグ移動
  - 期限の設定
  - 展開状態の保存
- **想定フロー**
  1) 人生/長期から短期へ分解
  2) 週間目標に落とし込む

### 約束
- **目的**: 自分や他者との約束を管理

## 3. 内省

### ミッション
- **目的**: 人生の方向性・存在意義を定義

### 価値観
- **目的**: 判断軸・優先順位を明確化

### Have / Do / Be
- **目的**: 欲しいもの / したいこと / ありたい姿の整理

### 演習（ワークシート系）

#### 価値観分析
- **目的**: 自分の価値観を深掘りする

#### 余命1年リスト
- **目的**: 優先順位を明確化する
- **形式**: 追加/削除できるリスト

#### あと100年人生リスト
- **目的**: 長期視点の可能性を広げる
- **形式**: 追加/削除できるリスト

#### 死ぬまでにやりたいこと
- **目的**: 人生の充実に向けたリスト化
- **形式**: 追加/削除できるリスト

#### 立場を変えて考える
- **目的**: 他者視点を取り入れて視野を広げる

#### 憧れの人物
- **目的**: ロールモデルを明確化する

#### 20年後の自分へインタビュー
- **目的**: 将来像を具体化する

#### 心に残った言葉・座右の銘
- **目的**: 行動指針になる言葉を残す

## 4. 人生設計のための使い方（推奨フロー）

1) **内省（ミッション/価値観/HaveDoBe/演習）**で「軸」を作る
2) **目標/約束**で長期→短期に落とす
3) **アクションプラン**でタスク化する
4) **週間計画**で実行と振り返りを回す
5) **Inbox/イシュー**で日々の気づきを取り込む

## 5. 設定

- **デフォルトタグ**: LifePlannerで作成/更新するMarkdownにfrontmatterの`tags`を付与します（カンマ区切り）。
- **タブ表示**: 不要なタブを非表示にできます。
